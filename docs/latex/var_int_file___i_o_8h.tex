\doxysubsection{Detailed Description}
File Input/\+Output for Variable-\/\+Length Words. 

VUF files are the end result of HyperC\textquotesingle{}s Compressor. Files are structured as a bit stream, where each value is stored as an i-\/bit value, where i can change between successive reads. \begin{DoxyNote}{Note}
